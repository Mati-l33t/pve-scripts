## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/nextcloud-exporter.sh)"
```

## Configuration

Config file:
```
/etc/nextcloud-exporter.env
```

## Web Interface

Default port: `9205`
