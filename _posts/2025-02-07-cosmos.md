## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/cosmos.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/etc/sysconfig/CosmosCloud
```

## Notes

> **INFO:** The file `/etc/sysconfig/CosmosCloud` is optional. If you need custom settings, you can create it yourself.

> **WARNING:** Requires FUSE support for mergerfs functionality. FUSE is enabled by default during installation.

## Web Interface

Default port: `80`
