## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/swizzin.sh)"
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Installation might take a long time if choosing to install many apps. Be patient.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Swizzin is a management suite, not a single application. Use the 'box' command inside the container to install/manage individual apps like rTorrent, Sonarr, etc. A full list can be found in documentation.</div>
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>It is very recommended to install at least the 'panel' for web access, and 'nginx' for easy access to other apps.</div>
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>WARNING: Installation sources scripts outside of Community Scripts repo. Please check the source before installing.</div>
</div>
