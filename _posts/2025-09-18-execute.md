## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/pve/execute.sh)"
```

## Notes

> **INFO:** Execute within the Proxmox shell.

> **INFO:** Non-Debian/Ubuntu containers will be skipped automatically.

> **WARNING:** Stopped containers will be started temporarily to run the command, then shut down again.
