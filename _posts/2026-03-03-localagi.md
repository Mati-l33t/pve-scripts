## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>This script builds LocalAGI from source (Go + Bun) and runs it as a systemd service.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>This Proxmox script runs LocalAGI in external-backend mode and does not provision local ROCm/NVIDIA runtimes.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>By default, LocalAGI is configured to call an OpenAI-compatible backend at <code>http://127.0.0.1:11434/v1</code> (Ollama-compatible) via <code>LOCALAGI_LLM_API_URL</code>.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>To use an external Ollama host, edit <code>/opt/localagi/.env</code> and set <code>LOCALAGI_LLM_API_URL=http://<ollama-host>:11434/v1</code>, then restart LocalAGI with <code>systemctl restart localagi</code>.</div>
</div>
