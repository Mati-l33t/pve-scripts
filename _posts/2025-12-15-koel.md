## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/koel.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Debian 13

## Default Credentials

| Username | Password |
|----------|----------|
| \`admin@koel.dev\` | \`KoelIsCool\` |

## Configuration

Config file:
```
/opt/koel/.env
```

## Notes

> **INFO:** Media files should be placed in /opt/koel_media

> **INFO:** Database credentials are stored in ~/koel.creds

> **INFO:** Music library is scanned hourly via cron job

## Web Interface

Default port: `80`
