## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/tracktor.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

| Username | Password |

Config file:
```
/opt/tracktor.env
```

## Notes

> **INFO:** Please check and update the '/opt/tracktor.env' file if using behind reverse proxy.
