## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/homepage.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 4096 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/homepage/config/
```

## Notes

> **INFO:** Configuration (bookmarks.yaml, services.yaml, widgets.yaml) path: `/opt/homepage/config/`

> **INFO:** Add additional allowed hosts to `/opt/homepage/.env`

## Web Interface

Default port: `3000`
