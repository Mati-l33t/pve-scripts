## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/agentdvr.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Ubuntu 24.04

## Configuration

Config file:
```
/opt/agentdvr/agent/Media/XML/config.json
```

## Web Interface

Default port: `8090`
