## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/scraparr.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/scraparr/config/config.yaml
```

## Notes

> **INFO:** Edit config file then restart the scraparr service: `systemctl restart scraparr`

## Web Interface

Default port: `7100`
