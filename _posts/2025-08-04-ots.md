## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/ots.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 512 MB &nbsp;|&nbsp; **Disk:** 3 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/ots/.env
```

## Notes

> **INFO:** When it is in used external please use it behind reverse proxy or create your own certificates

## Web Interface

Default port: `443`
