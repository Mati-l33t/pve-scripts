## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/kima-hub.sh)"
```
**CPU:** 4 core(s) &nbsp;|&nbsp; **RAM:** 8192 MB &nbsp;|&nbsp; **Disk:** 20 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/kima-hub/backend/.env
```

## Notes

> **INFO:** First user to register becomes the administrator.

> **WARNING:** Mount your music library to /music in the container.

> **INFO:** Audio analysis (mood/vibe detection) requires significant RAM (2-4GB per worker).

## Web Interface

Default port: `3030`
