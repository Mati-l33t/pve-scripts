## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/jellyfin.sh)"
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>With Privileged/Unprivileged Hardware Acceleration Support</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>FFmpeg path: /usr/lib/jellyfin-ffmpeg/ffmpeg</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>For NVIDIA graphics cards, you'll need to install the same drivers in the container that you did on the host. In the container, run the driver installation script and add the CLI arg --no-kernel-module</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Log rotation is configured in /etc/logrotate.d/jellyfin. To reduce verbosity, change MinimumLevel in /etc/jellyfin/logging.json to Warning or Error (disables fail2ban auth logging).</div>
</div>
