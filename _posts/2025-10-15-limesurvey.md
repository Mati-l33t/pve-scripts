## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/limesurvey.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 512 MB &nbsp;|&nbsp; **Disk:** 2 GB &nbsp;|&nbsp; **OS:** Debian 13

## Notes

> **INFO:** You will need to input database credentials into LimeSurvey installer. Use `cat ~/limesurvey.creds` inside LXC.

## Web Interface

Default port: `80`
