## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/resiliosync.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/etc/resilio-sync/config.json
```

## Notes

> **INFO:** After free registration, you will receive a license keyfile to your email address. Upload it into any LXC directory and select on first run.

## Web Interface

Default port: `8888`
