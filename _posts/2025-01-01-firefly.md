## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/firefly.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 2 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/firefly/.env
```

## Notes

> **INFO:** Data Importer is at `http://<LXC_IP>/dataimporter/`

> **INFO:** Database credentials: `cat ~/firefly.creds`

## Web Interface

Default port: `80`
