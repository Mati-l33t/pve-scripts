## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/pangolin.sh)"
```

  <span class="res-pill res-cpu">CPU: 2 cores</span>

  <span class="res-pill res-disk">Disk: 10 GB</span>

## Configuration

Config file:
```
/opt/pangolin/config/config.yml
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  Type `journalctl -u pangolin | grep -oP 'Token:\s*\K\w+'` into LXC console to get admin token which you will use to create admin account.
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  LXC has 4GB of RAM set initially for the build stage. After installation finishes, you can decrease the RAM allocated to 1024MB or 512MB even.
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  Make sure you edit `/opt/pangolin/config/config.yml` and change it to match your needs
</div>
