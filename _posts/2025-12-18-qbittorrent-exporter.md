## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/qbittorrent-exporter.sh)"
```

## Configuration

Config file:
```
/opt/qbittorrent-exporter.env
```

## Web Interface

Default port: `8090`
