## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/filebrowser-quantum.sh)"
```

## Default Credentials

| Username | Password |
|----------|----------|
| \`admin\` | \`helper-scripts.com\` |

## Configuration

Config file:
```
/usr/local/community-scripts/fq-config.yaml
```

## Web Interface

Default port: `8080`
