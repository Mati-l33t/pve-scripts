## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/miniflux.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Debian 13

## Default Credentials

| Username | Password |
|----------|----------|
| \`admin\` | \`randomly generated during installation process\` |

## Configuration

Config file:
```
/etc/miniflux.conf
```

## Notes

> **INFO:** Admin password available as `ADMIN_PASSWORD` in `~/miniflux.creds`

## Web Interface

Default port: `8080`
