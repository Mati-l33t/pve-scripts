## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/librenms.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/librenms/config.php and /opt/librenms/.env
```

## Notes

> **INFO:** After installation, the admin user credentials are saved in the file ~/librenms.creds inside the container.

## Web Interface

Default port: `80`
