## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/librenms.sh)"
```

Config file:
```
/opt/librenms/config.php and /opt/librenms/.env
```
