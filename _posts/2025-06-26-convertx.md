## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/convertx.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 4096 MB &nbsp;|&nbsp; **Disk:** 20 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/convertx/.env
```

## Notes

> **INFO:** Complete setup via the web interface at http://<container-ip>:3000. Create and secure the admin account immediately.
