## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/backrest.sh)"
```

## Configuration

Config file:
```
/opt/backrest/config/config.json | /opt/backrest/.env
```

<div class="info-callout">
