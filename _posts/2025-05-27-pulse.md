## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/pulse.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/etc/pulse
```

## Notes

> **INFO:** Create Proxmox-API-Token first: `https://github.com/rcourtman/Pulse?tab=readme-ov-file#creating-api-token`

> **INFO:** After installation, access the web interface to configure your Proxmox connection details through the built-in setup wizard

> **INFO:** Configure authentication in the Web UI => Settings => Security

## Web Interface

Default port: `7655`
