## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/pihole-exporter.sh)"
```

## Configuration

Config file:
```
/opt/pihole-exporter.env
```

## Web Interface

Default port: `9617`
