## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/cockpit.sh)"
```

  <span class="res-pill res-os">OS: Debian 12</span>

## Configuration

Config file:
```
/etc/cockpit/cockpit.conf
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  Set a root password if using autologin. This will be the Cockpit password. To set root password run `sudo passwd root`
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  If you plan on using 45Drives extension with NFS, you must setup LXC as privileged. Some features of 45Drives don't work on Debian 13, so Debian 12 must be used.
</div>
