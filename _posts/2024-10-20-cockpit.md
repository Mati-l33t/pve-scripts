## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/cockpit.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 12

## Configuration

Config file:
```
/etc/cockpit/cockpit.conf
```

## Notes

> **INFO:** Set a root password if using autologin. This will be the Cockpit password. To set root password run `sudo passwd root`

> **WARNING:** If you plan on using 45Drives extension with NFS, you must setup LXC as privileged. Some features of 45Drives don't work on Debian 13, so Debian 12 must be used.

## Web Interface

Default port: `9090`
