## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/tududi.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/tududi/backend/.env
```

## Notes

> **INFO:** Create users like this: `cd /opt/tududi` => `npm run user:create <email> <password>`

> **INFO:** Database location: `/opt/tududi-db`. Uploads: `/opt/tududi-uploads`

## Web Interface

Default port: `3002`
