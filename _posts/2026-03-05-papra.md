## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/papra.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 10 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/papra/.env
```

## Notes

> **INFO:** First visit will prompt you to create an account

> **INFO:** Tesseract OCR is pre-installed for all languages

> **INFO:** Documents are stored in /opt/papra/app-data/documents

> **INFO:** Ingestion folder available at /opt/papra/ingestion for automatic document import

> **WARNING:** Email functionality runs in dry-run mode by default

## Web Interface

Default port: `1221`
