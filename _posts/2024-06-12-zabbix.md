## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/zabbix.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 4096 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

## Default Credentials

| Username | Password |
|----------|----------|
| \`Admin\` | \`zabbix\` |

## Configuration

Config file:
```
/etc/zabbix/zabbix_server.conf
```

## Notes

> **INFO:** Database credentials: `cat ~/zabbix.creds`

> **INFO:** You can choose between Zabbix agent (classic) and agent2 (modern) during installation

> **INFO:** For agent2 the PostgreSQL plugin is installed by default; all plugins are optional

> **INFO:** If agent2 with NVIDIA plugin is installed in an environment without GPU, the installer disables it automatically
