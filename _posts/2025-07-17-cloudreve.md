## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/cloudreve.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 10 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/cloudreve/data/conf.ini
```

## Notes

> **WARN:** After Installation: Register your user -> Login -> Dashboard -> Accept Primary URL.

## Web Interface

Default port: `5212`
