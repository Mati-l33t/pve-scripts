## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/glances.sh)"
```

## Configuration

Config file:
```
/usr/local/share/doc/glances/glances.conf
```

## Notes

> **INFO:** Execute within an existing LXC Console (Debian / Ubuntu / Alpine supported)

## Web Interface

Default port: `61208`
