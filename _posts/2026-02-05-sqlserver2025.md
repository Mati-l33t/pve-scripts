## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/sqlserver2025.sh)"
```

## Notes

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>SQL Server (2025) SQLPAL is incompatible with Proxmox VE 9 (Kernel 6.12+) in LXC containers. Use a VM instead or the SQL-Server 2022 LXC.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>If you choose not to run the installation setup, execute: <code>/opt/mssql/bin/mssql-conf setup</code> in LXC shell.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>You can setup the admin account 'SA' during installation</div>
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>Make sure you disable the SA account if you intend to use this in production!</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Ubuntu 24.04 support requires SQL Server 2025 CU1 or later</div>
</div>
