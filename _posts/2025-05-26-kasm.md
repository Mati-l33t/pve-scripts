## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/kasm.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 8192 MB &nbsp;|&nbsp; **Disk:** 50 GB &nbsp;|&nbsp; **OS:** Debian 13

## Notes

> **WARNING:** WARNING: Installation sources scripts outside of Community Scripts repo. Please check the source before installing.

> **WARNING:** Kasm needs swap (on Proxmox host) and activated FUSE to be installed successfully!

> **INFO:** Show credentials: `cat ~/kasm.creds`

## Web Interface

Default port: `443`
