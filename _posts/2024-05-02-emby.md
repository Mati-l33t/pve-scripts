## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/emby.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Ubuntu 24.04

## Configuration

Config file:
```
/var/lib/emby/config/system.xml
```

## Notes

> **INFO:** With Privileged/Unprivileged Hardware Acceleration Support

Default port: `8096`
