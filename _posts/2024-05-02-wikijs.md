## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/wikijs.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 7 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/wikijs/config.yml
```
