## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/peanut.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 4096 MB &nbsp;|&nbsp; **Disk:** 7 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/etc/peanut/settings.yml
```

## Web Interface

Default port: `8080`
