## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/netbird.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 512 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/etc/netbird/config.json
```

## Notes

> **INFO:** The NetBird client (agent) allows a peer to join a pre-existing NetBird deployment. If a NetBird deployment is not yet available, there are both managed and self-hosted options available.

> **INFO:** After installation, enter the container and run `netbird` to view the commands.

> **INFO:** Use a Setup Key from your NetBird dashboard or SSO login to authenticate during setup or in the container.

> **INFO:** Check connection status with `netbird status`.
