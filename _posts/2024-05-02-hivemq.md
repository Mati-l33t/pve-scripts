## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/hivemq.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/hivemq/conf/config.xml
```

## Notes

> **INFO:** To check if HiveMQ is listening to the default port for MQTT `lsof -i :1883`

## Web Interface

Default port: `1883`
