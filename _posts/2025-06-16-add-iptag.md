## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/pve/add-iptag.sh)"
```

## Configuration

Config file:
```
/opt/iptag/iptag.conf
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Execute within the Proxmox shell</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Configuration: <code>nano /opt/iptag/iptag.conf</code>. iptag Service must be restarted after change. See here for full documentation: <code>https://github.com/community-scripts/ProxmoxVE/discussions/5790</code></div>
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>The Proxmox Node must contain ipcalc and net-tools. <code>apt-get install -y ipcalc net-tools</code></div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>You can execute the ip tool manually with <code>iptag-run</code></div>
</div>
