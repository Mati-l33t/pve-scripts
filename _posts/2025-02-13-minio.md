## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/minio.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 5 GB &nbsp;|&nbsp; **OS:** Debian 13

## Default Credentials

| Username | Password |
|----------|----------|
| \`minioadmin\` | \`None\` |

## Configuration

Config file:
```
/etc/default/minio
```

## Notes

> **INFO:** Application credentials: `cat ~/minio.creds`

Default port: `9001`
