## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/powerdns.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 1024 MB &nbsp;|&nbsp; **Disk:** 4 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/poweradmin/config/settings.php
```

## Notes

> **INFO:** For administrator credentials type: `cat ~/poweradmin.creds` inside LXC.

## Web Interface

Default port: `80`
