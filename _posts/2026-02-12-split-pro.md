## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVED/raw/main/ct/split-pro.sh)"
```

## Configuration

Config file:
```
/opt/split-pro/.env
```

## Notes

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>Before first use you must configure email credentials or authentication (OAuth/OIDC) provider in <code>/opt/split-pro/.env</code> and restart the service <code>systemctl restart split-pro</code>.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>Receipt uploads are stored in <code>/opt/split-pro_data/uploads</code></div>
</div>
