## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/plex.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 8 GB &nbsp;|&nbsp; **OS:** Ubuntu 24.04

## Configuration

Config file:
```
/var/lib/plexmediaserver/Library/Application Support/Plex Media Server/Preferences.xml
```

## Notes

> **INFO:** With Privileged/Unprivileged Hardware Acceleration Support
