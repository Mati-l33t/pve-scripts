## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/scanopy.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 3072 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/scanopy/.env, OIDC: /opt/scanopy/oidc.toml
```

## Notes

> **INFO:** To configure the integrated daemon after install is complete, use the `Create Daemon` menu in the UI and follow the instructions

> **INFO:** The integrated daemon config is located at `/root/.config/daemon/`

## Web Interface

Default port: `60072`
