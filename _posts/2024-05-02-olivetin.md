## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/olivetin.sh)"
```

## Configuration

Config file:
```
/etc/OliveTin/config.yaml
```

## Notes

> **INFO:** Configuration Path: `/etc/OliveTin/config.yaml`

## Web Interface

Default port: `1337`
