## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/booklore.sh)"
```
**CPU:** 3 core(s) &nbsp;|&nbsp; **RAM:** 3072 MB &nbsp;|&nbsp; **Disk:** 7 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/booklore_storage/.env
```

## Notes

> **INFO:** Starting Booklore (Web UI) may take up to 2 minutes after a restart or fresh installation.

## Web Interface

Default port: `6060`
