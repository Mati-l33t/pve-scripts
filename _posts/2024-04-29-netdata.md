## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/netdata.sh)"
```

## Configuration

Config file:
```
/etc/netdata/netdata.conf
```
