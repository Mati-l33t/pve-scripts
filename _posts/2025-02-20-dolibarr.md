## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/dolibarr.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 2048 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 12

## Configuration

Config file:
```
/usr/share/dolibarr/htdocs/conf/conf.php.example
```

## Notes

> **INFO:** Database credentials: `cat ~/dolibarr.creds`

## Web Interface

Default port: `80`
