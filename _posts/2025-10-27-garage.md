## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/garage.sh)"
```

## Configuration

Config file:
```
/etc/garage.toml
```

## Notes

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>The Garage configuration file is located at <code>/etc/garage.toml</code>. You can edit RPC and API bindings, tokens, and data directories there.</div>
</div>

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>Admin API runs by default on port <code>3903</code>, S3 API on port <code>3900</code>, Web UI on <code>3902</code>. Adjust firewall rules accordingly.</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>To view your generated tokens and RPC secret, check <code>~/garage.creds</code> after installation.</div>
</div>
