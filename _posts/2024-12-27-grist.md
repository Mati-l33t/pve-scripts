## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/grist.sh)"
```
**CPU:** 2 core(s) &nbsp;|&nbsp; **RAM:** 3072 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

## Configuration

Config file:
```
/opt/grist/.env
```
