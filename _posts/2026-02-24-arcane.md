## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/tools/addon/arcane.sh)"
```

## Default Credentials

| Username | Password |
|----------|----------|
| \`arcane\` | \`arcane-admin\` |

## Configuration

Config file:
```
/opt/arcane/.env
```

## Notes

> **INFO:** This is an addon script intended to be used on top of an existing Docker container.

Default port: `3552`
