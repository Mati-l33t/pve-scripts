## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/caddy.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 512 MB &nbsp;|&nbsp; **Disk:** 6 GB &nbsp;|&nbsp; **OS:** Debian 13

**Alpine install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/alpine-caddy.sh)"
```
**CPU:** 1 core(s) &nbsp;|&nbsp; **RAM:** 256 MB &nbsp;|&nbsp; **Disk:** 3 GB &nbsp;|&nbsp; **OS:** Alpine 3.23

## Configuration

Config file:
```
/etc/caddy/Caddyfile
```

## Notes

> **WARNING:** xcaddy needs to be updated manually after a caddy update!

> **INFO:** if you need an internal module run: `caddy add-package PACKAGENAME`

> **INFO:** if you need an external module run: `xcaddy build --with github.com/caddy-dns/cloudflare`

## Web Interface

Default port: `80`
