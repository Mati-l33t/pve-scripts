## Installation

**Default install:**
```bash
bash -c "$(wget -qLO - https://github.com/community-scripts/ProxmoxVE/raw/main/ct/caddy.sh)"
```

## Notes

<div class="warn-callout">
  <i class="fas fa-exclamation-triangle"></i>
  <div>xcaddy needs to be updated manually after a caddy update!</div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>if you need an internal module run: <code>caddy add-package PACKAGENAME</code></div>
</div>

<div class="info-callout">
  <i class="fas fa-info-circle"></i>
  <div>if you need an external module run: <code>xcaddy build --with github.com/caddy-dns/cloudflare</code></div>
</div>
